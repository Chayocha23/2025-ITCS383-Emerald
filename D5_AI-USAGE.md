# D5: AI-Usage Report

**Project:** SpaceHub (Co-Working Space Management System)

**Maintainer Group:** Bughair

## 🤖 AI Policy & Integration Summary
In this project phase, our group was officially authorized to utilize AI Coding Agents to enhance software maintenance efficiency. We prioritized transparency by systematically logging AI interactions across both analytical tasks and technical implementation. Every AI-generated output was cross-referenced with course materials and project requirements to ensure engineering integrity.

## 📋 Activities Log

### 1. Software Maintenance Analysis & Categorization (D3)
*   **Activity**: Analyzing and categorizing 8 formal Change Requests (CR) according to the ITCS383 maintenance models.
*   **AI Tool**: Claude.
*   **Prompt Example**: "Based on the ITCS383 lecture slides, if we are building a Native Android App for an existing system that previously had no mobile platform, should this be classified as Adaptive or Perfective maintenance? Provide a justification based on the provided definitions."
*   **Result**: **Claude** served as a theoretical consultant, helping the team accurately distinguish between environment adaptation and value-added improvements. This ensured that the D3 report was theoretically sound and aligned with the official grading criteria.

### 2. Native Android Application Development (D1)
*   **Activity**: Developing a native Android application using Kotlin and Jetpack Compose to replicate web functionalities.
*   **AI Tool**: Claude.
*   **Prompt Example**: "Generate a Kotlin class for Jetpack Compose that connects to the existing SpaceHub API (SLO10) and handles JWT authentication for mobile users."
*   **Result**: **Claude** assisted in generating UI structures and API integration logic, facilitating the delivery of a functional mobile client within the tight project timeline.

### 3. Backend Integration & CI/CD Pipeline (D1)
*   **Activity**: Adapting the Backend API for mobile support and configuring GitHub Actions with SonarCloud.
*   **AI Tool**: Claude / Gemini.
*   **Prompt Example**: "Help me debug the SonarCloud onboarding status. The analysis is running successfully in GitHub Actions, but the project is still stuck in 'onboarding' on the dashboard."
*   **Result**: Helped troubleshoot token configurations and YAML settings, ensuring the successful deployment of the CI/CD pipeline.

### 4. Unit Testing & Code Quality Improvement (D2)
*   **Activity**: Writing unit tests and refactoring code to ensure high maintainability and test coverage.
*   **AI Tool**: Claude / Gemini.
*   **Prompt Example**: "Refactor this booking logic (SLO2) to reduce complexity and write unit tests to ensure the new code test coverage exceeds 90%."
*   **Result**: The team successfully maintained the 'Clean as you Code' standard and achieved the mandatory >90% test coverage for all new code commits.
